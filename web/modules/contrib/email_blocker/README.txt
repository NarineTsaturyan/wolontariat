INTRODUCTION
------------

Email Blocker blocks all outgoing emails based on the configuration setting.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/email_blocker

 * To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/email_blocker


REQUIREMENTS
------------

No special requirements.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

* Settings is at `/admin/config/email_blocker/emailblockersettings`:

   - Block Emails:

     This will block all the outgoing emails throughout the site.
